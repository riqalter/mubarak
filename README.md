## Installation

## kalau memakai npm
```bash
  npx degit https://github.com/riqalter/mubarak
  npm install
  npm run dev
```
## kalau memakai yarn
```bash
  npx degit https://github.com/riqalter/mubarak
  yarn install
  yarn dev
```
